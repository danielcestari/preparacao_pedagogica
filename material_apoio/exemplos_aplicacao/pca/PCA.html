Anlise de componentes principais (Principal Component Analysis - PCA) Referncia: https://en.wikipedia.org/wiki/Principal_component_analysis

PCA um procedimento estatstico que usa transformao ortogonal para converter um conjunto de observaes de variveis possivelmente correlacionadas em um conjunto de valores de variveis linearmente no correlacionadas chamadas componentes principais.

Se h $n$ observaes com $p$ variveis, ento o nmero de componentes principais o mnimo entre $n-1$ e $p$. Esta transformao definida de maneira que a primeira componente principal tem a maior varincia possvel, ou seja, leva em conta a maior variabilidade possvel dos dados, e cada componente sucessiva tem por sua vez a maior varincia possvel sub as restries da transformao ortogonal das componentes restantes.

PCA de uma distribuio Gaussiana multivariada centrada em (1,3). Os vetores mostrados na figura so os autovetores da matriz de covarincia normalizada pela ras do autovalor correspondente.

PCA definido matematicamente como uma transformao ortogonal linear que transforma os dados para um novo sistema de coordenadas de modo que a maior varincia da projeo fique sobe o primeiro eixo coordenado (primeira componente principal), a segunda maior varincia sobe o segundo eixo principal, e assim por diante.

Considere a matriz de dados X, suas colunas tm mdia 0, e cada entrada das $n$ linhas representa uma observao diferente, e cada coluna representa um atributo das observaes.

Primeira Componente

Para maximizar a varincia, o vetor W precisa satisfazer:

$W_{(1)} = argmax \{ \frac{w^T X^T X w }{w^T w}\}$

Demais Componentes

A k-sima componente pode ser encontrada subtraindo as primeiras componentes de X:

$\hat{X_k} = X - \sum_{s=1}^{k-1} X w_{(s)} w^T_{(s)}$

e encontrando o prximo vetor $W_{(k)}$ atravs da expresso:

$W_{(s)} = argmax \{ \frac{w^T \hat{X^T_k} \hat{X_k} w }{w^T w}\}$.

A matriz completa com a componentes principais para a decomposio de X dada por: $T=XW$, onde W uma matriz com $p$ linhas e $p$ colunas cujas colunas so os autovalores da matriz $X^TX$.

%matplotlib inline
#print(__doc__)

import matplotlib.pyplot as plt

from sklearn import datasets
from sklearn.decomposition import PCA

iris = datasets.load_iris()

X = iris.data
y = iris.target
target_names = iris.target_names

pca = PCA(n_components=4)
X_r = pca.fit(X).transform(X)

print('Atributos do conjunto de dados %s\n'%(iris.feature_names))

# Percentage of variance explained for each components
print('Razo de varincia de cada componente principal: %s\n'
      % str(pca.explained_variance_ratio_))


print('Reduzindo a dimenso de 4 para 2 atributos\n')

pca = PCA(n_components=2)
X_r = pca.fit(X).transform(X)

plt.figure()
colors = ['navy', 'turquoise', 'darkorange']
lw = 2

for color, i, target_name in zip(colors, [0, 1, 2], target_names):
    plt.scatter(X_r[y == i, 0], X_r[y == i, 1], color=color, alpha=.8, lw=lw,
                label=target_name)
plt.legend(loc='best', shadow=False, scatterpoints=1)
plt.title('PCA of IRIS dataset')

plt.show()


print('\nPlotando grfico de disperso 2 a 2 entre os 4 atributos\n')
